What is Biopython?

Key Modules

Biopython is organized into submodules. Here are the ones you'll use most ‚Äî click any to learn more about it.

Bio.Seq

The core. Represents DNA, RNA, protein sequences. Supports transcription, translation, reverse complement.

Bio.SeqIO

Read/write FASTA, GenBank, FASTQ files. The pandas read_csv() of bioinformatics.

Bio.Entrez

Query NCBI databases (PubMed, GenBank, taxonomy) directly from Python.

Bio.Blast

Run BLAST searches (local or remote) and parse results programmatically.

Bio.AlignIO

Read/write multiple sequence alignments in various formats.

Bio.pairwise2

Local and global pairwise sequence alignment with scoring.

Bio.PDB

Parse PDB files. Analyze protein 3D structures, atoms, chains, residues.

Bio.Phylo

Build, parse, and visualize phylogenetic trees (Newick, Nexus formats).

Bio.motifs

Find and analyze sequence motifs ‚Äî useful for transcription factor binding sites.

Bio.PopGen

Population genetics statistics and coalescent simulations.

Real Use Cases + Code

Pick a use case to see how Biopython handles it.

Beginner Bio.Seq

Work with a DNA sequence ‚Äî transcribe it, translate it, find GC content.

from Bio.Seq import Seq
from Bio.SeqUtils import gc_fraction

dna = Seq("ATGGCCATTGTAATGGGCCGCTGAAAGGGTGCCCGATAG")

# Transcription: DNA ‚Üí mRNA
mrna = dna.transcribe() # AUGGCCAUUGUAAUGGGCCGCUGAAAGGGU...

# Translation: mRNA ‚Üí protein
protein = dna.translate() # MAIVMGR*KGAR*
protein_clean = dna.translate(to_stop=True) # Stops at first stop codon

# GC Content (0-1 scale)
gc = gc_fraction(dna) # e.g. 0.5384...
print(f"Protein: {protein_clean}")
print(f"GC content: {gc:.1%}")
Beginner Bio.SeqIO

Read a multi-sequence FASTA file and loop through records ‚Äî like pandas for sequences.

from Bio import SeqIO

# Parse a FASTA file ‚Äî returns an iterator of SeqRecord objects
for record in SeqIO.parse("sequences.fasta", "fasta"):
print(record.id) # sequence identifier
print(record.description) # full header
print(record.seq) # the sequence itself
print(len(record.seq)) # length

# Convert FASTA ‚Üí GenBank format in one line
SeqIO.convert("sequences.fasta", "fasta", "output.gb", "genbank")

# Load all into a dict (indexed by ID)
records = SeqIO.to_dict(SeqIO.parse("sequences.fasta", "fasta"))
my_gene = records["gene_id_123"]
Intermediate Bio.Blast

Run BLAST remotely on NCBI and parse results ‚Äî find similar sequences in the database.

from Bio.Blast import NCBIWWW, NCBIXML
from Bio import SeqIO

# Read a sequence from file
record = SeqIO.read("query.fasta", "fasta")

# Run BLASTn against NCBI nt database (takes a few seconds)
result_handle = NCBIWWW.qblast("blastn", "nt", record.seq)

# Parse results
blast_records = NCBIXML.parse(result_handle)
for blast_record in blast_records:
for alignment in blast_record.alignments[:5]: # top 5
hsp = alignment.hsps[0]
print(f"Hit: {alignment.title[:50]}")
print(f" E-value: {hsp.expect}, Score: {hsp.score}")

Intermediate Bio.PDB

Parse a PDB file and extract structural info ‚Äî chains, residues, atoms, distances.

from Bio.PDB import PDBParser, PPBuilder

parser = PDBParser(QUIET=True)
structure = parser.get_structure("protein", "1abc.pdb")

# Hierarchy: Structure ‚Üí Model ‚Üí Chain ‚Üí Residue ‚Üí Atom
model = structure[0] # first model
chain_a = model["A"] # chain A

for residue in chain_a.get_residues():
print(residue.resname, residue.id[1]) # e.g. ALA 23

# Get all CA (alpha carbon) atom positions
ca_atoms = [atom for atom in structure.get_atoms() if atom.name == "CA"]
print(f"{len(ca_atoms)} residues in structure")

Quick Knowledge Check

5 questions to see how much is sticking. No pressure ‚Äî it's just for you.

1. Which Biopython module would you use to read a FASTA file?

‚úÖ SeqIO is the I/O module for sequence files ‚Äî think of it like pandas' read_csv() but for bioinformatics formats.

2. What does dna.translate() return?

‚úÖ translate() goes DNA ‚Üí protein directly. Use transcribe() first if you want the mRNA intermediate step explicitly.

3. What is a SeqRecord?

‚úÖ SeqRecord wraps a Seq with extra info: ID, description, features, annotations. It's what SeqIO.parse() returns.

4. Which function do you use to search NCBI databases?

‚úÖ esearch() searches and returns IDs. Then use efetch() to actually download those records. qblast() is specifically for BLAST.

5. In Bio.PDB, what is the hierarchy of a protein structure?

‚úÖ It's SMCRA: Structure ‚Üí Model ‚Üí Chain ‚Üí Residue ‚Üí Atom. This nested structure mirrors how PDB files are organized.
